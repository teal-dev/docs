---
title: Attaching metatags
description: "Setting up metatags for advanced reporting."
icon: #"tags"
---

Teal allows you to attach metatags to transactions and line entries to allow advanced reporting. For example, transactions can be tagged to a location, employee, or project. This allows you to generate reports on specific tags, for example, generating an income statement for a specific location or project.

This feature is currently in development. Please contact us if you would like to use this feature before Q4 2023.
