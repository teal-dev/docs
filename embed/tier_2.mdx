**Tier 2** provides users with a full accounting solution. Users are able to view and manage their Income Statement and Balance Sheet, as well as create manual journal entries for off platform transactions.

## Income Statement

You can use the [Get Income Statement endpoint](/api-reference/reports/get_income_statement) in order to get the data required for producing an income statement.

## Balance Sheet

You can use the [Get Balance Sheet endpoint](/api-reference/reports/get_balance_sheet) in order to get the data required for producing a balance sheet.

## Accounting Package Export

An accounting package can be exported for a user as an Excel file. The file is intended to be used by an accountant.
